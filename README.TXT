Project Name: File System Analyzer  
Author: Yasrib Yasir Farook

Description:  
The File System Analyzer program provides a way to analyze directories on a file system. 
It counts the number of files in a directory and its subdirectories, lists all files, and searches for a specific file. 
It employs recursive methods to navigate and process directories.  

Classes Included:  
1. FileSystemAnalyzer.java
   - The main class that contains methods to:
     - Count files in a directory.
     - Display all files in a directory and its subdirectories.
     - Search for a specific file and retrieve its path if found.
